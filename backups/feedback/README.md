# Feedback Backups

This folder stores manual CSV exports from Formspree.

## How to export
1. Go to https://formspree.io (log in)
2. Open your form → Submissions tab
3. Click "Export CSV"
4. Save the file here as: feedback-backup-YYYY-MM.csv

## Why this matters
If Formspree ever becomes unavailable, this folder preserves your submission history.
Files in this folder are gitignored and will not be pushed to GitHub.
